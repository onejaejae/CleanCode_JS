### var를 지양하자

---

<br>

✔ var => 함수 스코프

✔ let, const => 블록 스코프, TDZ(Temporal Dead Zone) => 안전한 코드 작성 가능

<br>

```js
var name = "이름";
var name = "이름2";
var name = "이름3";
var name = "이름4";
console.log(name); // 이름 4

let name = "이름";
let name = "이름2";
let name = "이름3";
// SyntaxError: Identifier 'name' has already been declared

const name = "이름";
const name = "이름2";
const name = "이름3";
// SyntaxError: Identifier 'name' has already been declared
```

📌 let, const의 차이점은 `재할당` 여부
